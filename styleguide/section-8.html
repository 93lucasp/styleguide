8.2 #permalink Buttons modification

For the modification section we use 3 differents type of buttons that we structured in this way.

  • For the editing we use the glyphicon-pencil icon.
  • To add a folder we use the glyphicon-plus icon.
  • To check a folder we use the glyphicon-file icon.
    How we can see we don't use a real button tag but we add the btn class to the link and it will takes the apparence of a button!
Example 
<a class="glyphicon glyphicon-size glyphicon-pencil btn btn-primary btn-sm" href=""> </a>
<a class="glyphicon glyphicon-size glyphicon-plus btn btn-primary btn-sm" href=""> </a>
<a class="glyphicon glyphicon-size glyphicon-file btn btn-primary btn-sm" href=""> </a>
