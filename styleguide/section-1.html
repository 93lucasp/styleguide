1 #permalink Grid System

We use the Bootstrap's grid system and it is an important concept to understand.
How the grid system works?
We have two important components: ROW and COLUMN.

  • Rows classes are horizontal groups of columns that allow you to align correctly the columns. The content should be the columns, and only columns may be immediate children of rows.
  • Columns classes indicate the number of columns you would like to use out of the possible 12 per row. Column widths are set in percentages, so theyâ€™re always fluid or sized relative to their parent element. Columns have some horizontal padding by default.

How the grid system works across multiple devices?
With a handy table with the differents classes, it will be responsive under the following break point.

Parameters:
  • .col-xs-
    Extra small (Auto), it will try to keep 12 colums align also in small devices.
  • .col-sm-
    Small (544px).
  • .col-md-
    Medium (768px).
  • .col-lg-
    Large (992px).
  • .col-xl-
    Extra large (1200px).
